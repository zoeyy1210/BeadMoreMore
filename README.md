# 豆子多多 - 拼豆图纸生成器

一款基于浏览器的拼豆图纸生成工具，支持上传图片、选择拼豆品牌调色板，一键生成专业拼豆图纸与颜色统计清单。

## 在线体验

直接双击 `index.html` 文件即可在浏览器中打开使用，无需安装任何依赖。

## 功能特性

### 图纸生成模式
- **图片上传**：支持 JPG、PNG、WebP 格式，可拖拽上传或点击选择
- **多品牌支持**：内置 MARD、Hama、Perler、Artkal（S/R/C/A）、Nabbi 等主流拼豆品牌调色板
- **自定义网格**：支持 10-200 精度调节，控制图纸细腻程度
- **智能配色**：基于 RGB 欧氏距离算法，自动匹配最接近的拼豆颜色
- **图纸编辑**：生成图纸后支持编辑功能
  - 点选修改：点击任意像素格替换颜色
  - 框选批量替换：拖拽框选区域后统一替换颜色
  - 撤销/重做：支持 50 步历史记录（Ctrl+Z / Ctrl+Y）
- **颜色统计**：自动生成每种颜色的使用数量、占比百分比，支持按数量或名称排序
- **一键复制**：快速复制颜色统计清单，方便对照采购或整理豆子
- **悬浮提示**：鼠标悬停像素格可查看对应颜色名称、色值与坐标位置
- **清晰网格线**：双层网格系统，每格细线 + 每 5 格粗线，便于阅读和打印

### 自由画图模式
- **空白画布**：无需上传图片，直接创建空白画布自由绘制
- **画笔工具**：在任意像素格上绘制所选颜色
- **橡皮擦**：擦除已绘制的像素
- **填充工具**：一键填充闭合区域
- **形状工具**：支持绘制直线、矩形、圆形
- **调色板**：使用品牌调色板选择颜色，确保颜色可购买
- **撤销/重做**：支持 50 步历史记录

## 技术实现

- **纯前端实现**：HTML + CSS + JavaScript，零依赖，无需构建工具
- **Canvas API**：用于像素画渲染与网格线绘制
- **CSS 设计系统**：基于 HSL 色彩空间的自定义属性（CSS Custom Properties），支持统一的色彩、间距、阴影、过渡规范
- **色彩匹配算法**：RGB 欧氏距离（跳过 sqrt 优化性能）
- **编辑功能实现**：
  - 撤销/重做：基于命令模式的历史栈（50 步）
  - 框选：鼠标拖拽计算选区坐标，批量替换像素数据
  - 填充算法：四连通洪水填充（Flood Fill）
  - 形状绘制：Bresenham 直线算法 + 几何图形绘制
- **响应式布局**：适配桌面端与移动端
- **现代 UI 设计**：胶囊分段控件 Tab、玻璃拟态效果、平滑过渡动画

## 使用方法

### 图纸生成模式

1. 用浏览器打开 `index.html`
2. 点击顶部「📐 图纸生成」标签
3. 上传一张图片（拖拽或点击上传区域）
4. 选择拼豆品牌（默认为 MARD）
5. 设置网格精度（默认 50，范围 10-200）
6. 点击「生成图纸」按钮
7. 使用工具栏进行编辑（查看/编辑/框选模式）
8. 查看颜色统计信息

### 自由画图模式

1. 点击顶部「🎨 自由画图」标签
2. 选择拼豆品牌和网格精度
3. 点击「新建画布」按钮
4. 使用工具栏选择画笔、橡皮擦、填充或形状工具
5. 在画布上自由绘制
6. 查看实时颜色统计

| 令牌 | 色值 | 用途 |
|------|------|------|

| Surface | HSL(0, 0%, 100%) | 卡片背景 |
| Background | HSL(36, 33%, 97%) | 页面底色 |
| Border | HSL(20, 6%, 88%) | 边框色 |
